# 2nd Hand Market

A React SPA for browsing products, filtering items, adding products to favorites, and reserving products.

## Functionality

- product catalog browsing;
- product search;
- filtering by categories and parameters;
- sorting by price;
- adding products to Favorites;
- adding products to Reserved;
- viewing Reserved products;
- product details page;
- profile page with user statistics;
- login and registration flow.

## Dependencies

- React
- React DOM
- React Router DOM
- Redux Toolkit
- React Redux
- Webpack
- Webpack CLI
- Webpack Dev Server
- Babel
- Babel Loader
- CSS Loader
- Style Loader
- HTML Webpack Plugin
- DummyJSON API

## Installing dependencies

```bash
npm install react react-dom react-router-dom @reduxjs/toolkit react-redux
npm install -D webpack webpack-cli webpack-dev-server html-webpack-plugin
npm install -D babel-loader @babel/core @babel/preset-env @babel/preset-react @babel/plugin-transform-modules-commonjs
npm install -D css-loader style-loader
```

### Running the project
```bash
npm start
```

### Building the project
```bash
npm run build
```

### The production build will be created in the folder:
```bash
dist
```
